Введите API-ключ Яндекс.Карт

Как получить API-ключ?

Для работы приложения необходим API-ключ от Яндекс.Карт. Вы можете получить его бесплатно на странице разработчика Яндекс.

О проекте

Это минималистичное веб-приложение для массового геокодирования адресов с использованием API Яндекс.Карт. Оно позволяет быстро получить географические координаты (широту и долготу) для большого списка адресов.

Исходный код проекта доступен на GitHub.
